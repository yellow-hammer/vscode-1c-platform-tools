# ER-диаграммы метаданных

Интерактивная схема связей объектов конфигурации прямо в IDE: выберите объекты — и увидите, как они связаны через реквизиты, табличные части, движения и другие зависимости.

![ER-диаграмма метаданных](../walkthrough/images/step11.png)

## Как открыть

- Кнопка **ER-диаграмма** в панели **Метаданные 1С**, или
- контекстное меню любого объекта дерева метаданных — конструктор откроется с выбранным объектом и его прямыми связями.

## Конструктор

- **Поиск** добавляет объекты на диаграмму.
- **Глубина**: «Выбранные» — только выбранные объекты, «Прямые» — с их прямыми связями.
- **Фильтры видов связей** — по типам зависимостей (реквизиты, ТЧ, движения и т.д.).

## Экспорт

Диаграмма экспортируется в **Mermaid**, **Draw.io**, **SVG** и **PNG**. По умолчанию результат сохраняется в `docs/schemas` проекта.

Настройки: `1c-platform-tools.metadata.er.exportDir` — каталог экспорта, `1c-platform-tools.metadata.er.defaultExportFormat` — формат по умолчанию.
