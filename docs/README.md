# Документация 1C: Platform Tools

Правила каталога: `docs/` — пользовательские руководства, по одному документу на функцию расширения. Краткий обзор возможностей и обязательная настройка — в [README проекта](../README.md), правила разработки и релизов — в [CONTRIBUTING.md](../CONTRIBUTING.md).

## Руководства по функциям

- **[Инструменты 1С](tools.md)** — дерево команд vanessa-runner: разделы, избранное, настройки путей.
- **[Проекты 1С](projects.md)** — поиск проектов, избранное и теги, быстрое переключение, синхронизация.
- **[Артефакты 1С](artifacts.md)** — дерево артефактов проекта и действия сборки/разбора.
- **[Метаданные 1С](metadata.md)** — дерево метаданных, создание и изменение объектов, первый запуск и настройки.
- **[ER-диаграммы](er-diagrams.md)** — конструктор схем связей метаданных и экспорт.
- **[Список дел](todo.md)** — поиск TODO-комментариев, фильтры и настройки.
- **[Отладка 1С](debug.md)** — настройка, запуск, точки останова (условия, логирование, автоперенос), расширения и внешние обработки, переменные и изменение значений, вычисление выражений, замер производительности.
- **[AI и MCP](ai-mcp.md)** — навыки для агентов, файл-триггер, MCP-сервер.
- **[Docker и ibcmd](docker.md)** — выполнение команд в контейнере, сборка образов, GitHub Codespaces.

Пошаговое знакомство со всеми панелями — walkthrough **Начало работы с 1C: Platform Tools** (Help → Welcome) или каталог [walkthrough](../walkthrough).
