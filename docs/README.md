# Документация проекта 1C: Platform tools

Добро пожаловать в документацию проекта! Здесь вы найдете всю необходимую информацию для разработки и использования расширения.

## Содержание

### Для разработчиков

- **[VS Code API - Паттерны использования](vscode-api.md)**  
  Подробное руководство по использованию VS Code API в расширении. Включает примеры работы с workspace, уведомлениями, командами, терминалами, TreeView и BaseCommand.

- **[Паттерны работы с vrunner](vrunner-patterns.md)**  
  Документация по использованию vrunner (vanessa-runner) в расширении. Описывает работу с BaseCommand, VRunnerManager, выполнение команд в терминале, работу с путями, утилиты для команд и обработку ошибок.

- **[Настройка CI/CD](CI_CD_SETUP.md)**  
  Пошаговая инструкция по настройке автоматической сборки и публикации расширения в VS Code Marketplace через GitHub Actions.

### Для пользователей

- **[README проекта](../README.md)**  
  Основная документация проекта с описанием возможностей, установки и использования.

- **[Руководство для контрибьютеров](../CONTRIBUTING.md)**  
  Информация для тех, кто хочет внести вклад в проект.

## Быстрый старт

1. Прочитайте [README проекта](../README.md) для общего понимания
2. Изучите [VS Code API паттерны](vscode-api.md) для работы с API
3. Ознакомьтесь с [паттернами vrunner](vrunner-patterns.md) для работы с командами 1С
4. Следуйте [руководству для контрибьютеров](../CONTRIBUTING.md) для начала разработки
