# Артефакты 1С

Панель **Артефакты 1С** в Проводнике показывает поддерживаемые файлы проекта: конфигурации (`*.cf`), расширения (`*.cfe`), внешние обработки (`*.epf`), отчёты (`*.erf`), feature-файлы (`*.feature`) и соответствующие каталоги исходников.

![Панель «Артефакты 1С»](../walkthrough/images/step4.png)

## Действия

- Из дерева: **Открыть**, **Собрать**, **Разобрать**; для feature-файлов — **Запустить** (Vanessa-Automation). В контекстном меню — те же команды и **Удалить**.
- При открытии бинарного файла (`.cf`, `.cfe`, `.epf`, `.erf`) в редакторе доступна кнопка **Разобрать**.

Сборка и разбор выполняются командами vanessa-runner в терминале — по тем же путям, что и команды дерева **Инструменты 1С** (`1c-platform-tools.paths.*`).

## Настройки

`1c-platform-tools.artifacts.exclude` — сегменты пути, исключаемые при сканировании артефактов (например `oscript_modules`, `build`).
