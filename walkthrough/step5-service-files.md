# Шаг 5 — Служебные файлы

![Скриншот](images/placeholder.svg)

Служебные файлы проекта хранят параметры подключения к информационной базе, настройки отладки и исключения для Git.

**env.json** — строка подключения к ИБ (`--ibconnection`), используется командами vrunner. **.gitignore** и **.gitattributes** помогают корректно работать с репозиторием. Файлы `tools/*` хранят параметры тестов и синтаксического контроля.

Раздел **Служебные файлы** в дереве «Инструменты 1С» создаёт эти файлы из шаблонов: пункт **Создать служебные файлы** открывает список с пометкой наличия, **Базовый набор** создаёт `.gitignore`, `.gitattributes` и `env.json` одним действием. Подробнее — [Служебные файлы](../docs/service-files.md) и [Профили запуска](../docs/launch-profiles.md).
