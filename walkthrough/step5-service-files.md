# Шаг 5 — Служебные файлы

![Скриншот](images/placeholder.svg)

Служебные файлы проекта хранят параметры подключения к информационной базе, настройки отладки и исключения для Git.

**Осн. профиль запуска** — строка подключения к ИБ и параметры запуска; формат файла зависит от версии vanessa-runner (`env.json` для 2.x, `autumn-properties.json` для 3.x). **.gitignore** и **.gitattributes** помогают корректно работать с репозиторием. Файлы `tools/*` хранят параметры тестов и синтаксического контроля.

Раздел **Служебные файлы** в дереве «Инструменты 1С» создаёт эти файлы из шаблонов: пункт **Создать служебные файлы** открывает список с пометкой наличия, **Базовый набор** создаёт `.gitignore`, `.gitattributes` и основной профиль запуска одним действием. Подробнее — [Служебные файлы](../docs/service-files.md) и [Профили запуска](../docs/launch-profiles.md).
