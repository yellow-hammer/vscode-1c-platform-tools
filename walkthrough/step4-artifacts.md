# Шаг 4 — Панель артефактов

![Скриншот](images/step4.png)

Панель **«Артефакты 1С»** на боковой панели (см. Проводник) показывает конфигурации, расширения, внешние обработки и отчёты проекта.

Для бинарных файлов (`.cf`, `.cfe`, `.epf`, `.erf`) доступна кнопка **«Разобрать»** в панели редактора. Из дерева артефактов: **Открыть**, **Собрать** или **Разобрать**. В контекстном меню — те же команды и **Удалить**. Тесты (включая сценарии Vanessa из `.feature`) запускаются из панели тестирования.
